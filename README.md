A secure and user-friendly E-Wallet Web Application developed using Java, Spring Boot, Maven, and Thymeleaf. The application allows users to create accounts, securely log in, manage wallet balances, transfer money between users, and view transaction history through an intuitive web interface.

Features
👤 User Registration
🔐 Secure Login & Authentication
💰 Wallet Balance Management
💸 Money Transfer Between Users
📜 Transaction History
🗄️ Database Integration using Spring Data JPA
🌐 Responsive User Interface
⚡ Fast and Lightweight Spring Boot Application
